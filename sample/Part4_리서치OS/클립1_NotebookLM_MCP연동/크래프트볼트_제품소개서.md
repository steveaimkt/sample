## USP (Unique Selling Points)

### 1. 압도적 경량 — 1.8kg
- 동급 8인치 전기톱 평균 무게: 2.5~3.2kg
- 크래프트볼트: 1.8kg → **동급 대비 28~44% 가벼움**
- 한 손 작업 가능 → 초보자·여성 사용자도 쉽게 사용

### 2. 브러시리스 모터 — 강력하고 오래감
- 브러시 모터 대비 수명 3배, 효율 20% 향상
- 최대 절단력: 직경 15cm 참나무 절단 가능
- 동급 브러시 모터 제품 대비 배터리 사용 시간 25% 더 오래감

### 3. 완제품 출고 — 조립 불필요
- 개봉 즉시 사용 가능 (체인바 장착 완료 상태로 배송)
- 경쟁사 대부분: 체인 장착·텐션 조절 직접 해야 함
- "전기톱 처음인데 조립이 걱정" → 걱정 제로

### 4. 배터리 2개 기본 제공
- 배터리 1개 연속 사용: 약 40분
- 2개 교대 사용 시: 약 1시간 20분 연속 작업 가능
- 충전 중에도 다른 배터리로 작업 지속

### 5. 1년 무상 A/S
- 구매일로부터 1년간 무상 수리
- 전화 상담: 평일 09:00~18:00
- 택배 수리: 왕복 배송비 무료
