### 클립1. 워크스페이스 세팅 및 권한 설정

| 파일 | 형식 | 설명 | 사용법 |
|------|------|------|--------|
| `폴더구조_예시.md` | MD | input/output/template 등 폴더 트리와 권한 설명 | 읽고 내 프로젝트 폴더를 동일하게 구성 |

| `gitignore_예시.txt` | TXT | .env, .DS_Store 등 보호 대상 목록 | `.gitignore`로 이름 변경하여 프로젝트 루트에 배치 |

**실습 프롬프트**
```
이 MCP 설정 파일을 확인하고, 현재 연동 가능한 서비스 목록을 정리해줘.
```

## 파일 유형별 범례

| 아이콘 | 유형 | 용도 |
|--------|------|------|
| CSV | 데이터 | 클로드에 전달하여 분석하는 **입력 데이터** |
| PDF | 소스 | NotebookLM에 업로드하는 **외부 소스** |
| JSON | 입력/템플릿 | 피그마 플러그인에 넣거나 내 제품으로 채우는 **구조화된 입력** |
| MD (읽기) | 가이드 | 개념 설명·절차 안내·프롬프트 모음 — **먼저 읽고 이해** |
| MD (예시) | 결과물 샘플 | "이런 형태로 나와요" — **결과 품질 확인용** |
| ENV/JSON/TXT (설정) | 설정 템플릿 | 프로젝트 세팅 시 복사하여 사용하는 **설정 파일** |
