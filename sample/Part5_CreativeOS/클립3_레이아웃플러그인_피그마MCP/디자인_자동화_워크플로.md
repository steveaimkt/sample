# 레이아웃 플러그인 & 피그마 MCP 워크플로

## 전체 파이프라인

```
NotebookLM (리서치)
    ↓ Pain Point, 공감 키워드, 시장 데이터
클로드 (기획)
    ↓ 상세페이지 기획안
클로드 (JSON 변환)
    ↓ 기획안_JSON.json
레이아웃 플러그인 (피그마)
    ↓ 상세페이지 레이아웃 자동 구축
피그마 MCP (디자인 자동화)
    ↓ 디자인 시안 완성
```

---

## Step 1: 클로드에서 기획안 JSON 생성

### 프롬프트 예시
```
비타민C 세럼의 Pain Point 리서치 결과를 바탕으로
상세페이지 기획안을 JSON 형식으로 만들어줘.

참고 파일:
- sample/Part5_CreativeOS/클립1_PainPoint_심층리서치/pain_point_리서치_예시.md
- sample/Part5_CreativeOS/클립2_AgentTeams_Debate/agent_teams_debate_예시.md

JSON 구조:
- product: 제품 정보
- sections: 섹션별 레이아웃, 콘텐츠, 스타일
```

---

## Step 2: 레이아웃 플러그인에 JSON 입력

### 사용 방법
1. 피그마에서 레이아웃 플러그인 실행
2. JSON 파일 업로드 또는 붙여넣기
3. 자동으로 섹션별 프레임 생성
4. 텍스트·이미지 플레이스홀더 배치

---

## Step 3: 피그마 MCP로 디자인 자동화

### 안티그래비티에서 호출 예시
```
피그마 MCP를 사용해서
기획안_JSON.json 기반으로 상세페이지 디자인 시안을 만들어줘.

스타일 가이드:
- 메인 컬러: #1a1a2e
- 서브 컬러: #e94560
- 폰트: Pretendard
- 톤: 깔끔하고 신뢰감 있는
```
