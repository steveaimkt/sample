# 워크스페이스 폴더 구조 예시

```
프로젝트_루트/
├── CLAUDE.md          # 에이전트 지시서 (역할·규칙·톤 정의)
├── input/             # 읽기 전용 — 원본 데이터
│   ├── 광고데이터.csv
│   ├── 고객리뷰.csv
│   └── 매출데이터.xlsx
├── output/            # 결과물 저장
│   ├── 분석리포트.md
│   ├── 상세페이지.html
│   └── 히어로이미지.png
├── template/          # 작업 양식
│   ├── 리포트_템플릿.md
│   └── 상세페이지_템플릿.html
├── config/            # 설정 파일
│   └── mcp_settings.json
├── skills/            # 스킬 정의
│   └── brand-guideline/
│       └── SKILL.md
└── sample/            # 데모·테스트용 샘플 데이터
```

## 폴더별 역할 요약

| 폴더 | 권한 | 용도 |
|------|------|------|
| `input/` | 읽기 전용 | 원본 데이터 보관. 에이전트가 수정하지 않음 |
| `output/` | 읽기/쓰기 | 에이전트가 생성한 모든 결과물 저장 |
| `template/` | 읽기 전용 | 작업 시 참조하는 양식 |
| `config/` | 읽기 전용 | MCP 서버 설정, API 키 참조 |
| `skills/` | 읽기 전용 | 브랜드 가이드라인, 페르소나 정의 |
| `sample/` | 읽기 전용 | 테스트·데모용 데이터 |
