# Editor View 예시 프롬프트

## Editor View란?
내가 **직접 편집하면서** AI와 함께 세부 수정·미세 조정하는 모드.
특정 문장, 카피, 코드의 일부분만 다듬고 싶을 때 사용.

---

## 예시 1: 카피 톤 조정
```
이 헤드라인을 좀 더 긴급감 있게 바꿔줘:
"비타민C 세럼으로 피부를 관리하세요"
→ 더 강렬하게, 고객이 바로 클릭하고 싶게
```

## 예시 2: 특정 섹션 수정
```
상세페이지의 "고객 후기" 섹션에서
3번째 후기의 톤을 좀 더 자연스럽게 다듬어줘
```

## 예시 3: 코드 일부 수정
```
output/상세페이지.html에서
히어로 배너의 배경색을 #1a1a2e에서 #16213e로 변경해줘
```

---

## Editor View 선택 가이드
| 상황 | 적합 여부 |
|------|----------|
| 카피 한 줄 톤 조정 | ✅ 적합 |
| HTML/CSS 미세 수정 | ✅ 적합 |
| 전체 리포트 작성 | ❌ Manager View 추천 |
| 데이터 분석 전체 과정 | ❌ Manager View 추천 |
