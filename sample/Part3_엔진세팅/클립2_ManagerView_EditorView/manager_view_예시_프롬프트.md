# Manager View 예시 프롬프트

## Manager View란?
큰 지시만 내리면 에이전트가 **알아서 계획 → 실행 → 검증**하는 모드.
복잡한 분석이나 리포트 생성처럼 전체 흐름을 맡기고 싶을 때 사용.

---

## 예시 1: 광고 데이터 분석
```
쿠팡 광고 데이터를 분석해서 12월 최적화 방안을 리포트로 만들어줘.
input/쿠팡_광고_키워드.csv 파일을 참고해.
```

## 예시 2: 경쟁사 리뷰 분석
```
input/고객리뷰_500개.csv를 분석해서
주요 불만 사항 TOP 5와 만족 포인트 TOP 5를 정리해줘.
```

## 예시 3: 시장 분석 리포트
```
비타민C 세럼 시장의 경쟁사 리뷰와 상세페이지를 분석해서
시장 인사이트 리포트를 만들어줘.
```

---

## Manager View 선택 가이드
| 상황 | 적합 여부 |
|------|----------|
| 전체 분석 리포트가 필요할 때 | ✅ 적합 |
| 데이터 기반 인사이트 도출 | ✅ 적합 |
| 카피 한 줄만 수정할 때 | ❌ Editor View 추천 |
| 특정 문장 톤 미세 조정 | ❌ Editor View 추천 |
