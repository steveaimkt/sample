# Plan Mode 실행 예시

## 사용 방법
사용자가 `plan` 또는 `플랜`을 입력하면 Plan Mode에 진입합니다.

---

## 예시: "쿠팡 광고 분석해줘"에 대한 Plan Mode 출력

```
## 작업 계획

**목표**: 쿠팡 광고 키워드 데이터를 분석하여 12월 최적화 액션 플랜 도출

### Step 1: 데이터 확인
- 작업 내용: input/쿠팡_광고_키워드.csv 파일 구조 및 데이터 품질 확인
- 참조 파일: input/쿠팡_광고_키워드.csv
- 산출물: 없음 (분석 준비 단계)

### Step 2: 키워드 성과 분석
- 작업 내용: 키워드별 노출수, 클릭수, 전환율, ROAS 계산 및 순위 정렬
- 참조 파일: input/쿠팡_광고_키워드.csv
- 산출물: `output/키워드_성과_분석.md`

### Step 3: 저성과 키워드 식별
- 작업 내용: ROAS 100% 미만, CTR 1% 미만 키워드 분류 및 원인 분석
- 참조 파일: Step 2 산출물
- 산출물: `output/저성과_키워드_리스트.md`

### Step 4: 최적화 액션 플랜 도출
- 작업 내용: 키워드 추가/제거/입찰가 조정 등 구체적 실행 방안 정리
- 참조 파일: Step 2, 3 산출물
- 산출물: `output/12월_광고_최적화_액션플랜.md`

---
**확인 필요 사항**:
- 분석 기간을 12월 전체로 할까요, 아니면 특정 주차를 지정할까요?
- ROAS 기준선을 100%로 설정해도 괜찮을까요?

**예상 리스크**:
- CSV 파일에 빈 값(결측치)이 있을 수 있음 → 결측치 처리 방식 사전 확인 필요
```

---

## Plan Mode가 유리한 상황
| 상황 | Plan Mode 필요도 |
|------|-----------------|
| 3단계 이상의 복잡한 작업 | ⭐⭐⭐ 필수 |
| 여러 파일을 참조하는 분석 | ⭐⭐⭐ 필수 |
| 단순 카피 수정 | ⭐ 불필요 |
| 한 줄 질문 답변 | ⭐ 불필요 |
